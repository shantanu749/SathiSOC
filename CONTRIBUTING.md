# Contributing to SathiSOC Lite

Thanks for helping improve SathiSOC Lite. This project is intended to stay public-safe, useful for small teams, and easy to adapt.

## Development

```bash
npm install
npm run dev
```

Before opening a pull request:

```bash
npm run lint
npm run build
```

## Contribution Guidelines

- Do not add real company data, private endpoints, credentials, customer records, or secrets.
- Keep mock events documentation-safe.
- Prefer small, focused pull requests.
- Keep components reusable and typed.
- Add notes in the README or docs when a change affects backend integration.

## Useful Areas

- Backend API integration
- Detection rule engine
- Accessibility improvements
- Export/reporting workflows
- SOC playbook content
- Tests and CI coverage
