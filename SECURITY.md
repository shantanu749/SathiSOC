# Security Policy

## Supported Versions

SathiSOC Lite is an early open-source project. Security fixes should target the latest version on the main branch.

## Reporting a Vulnerability

Please do not open a public issue for vulnerabilities that expose sensitive implementation details. Instead, contact the project maintainer privately and include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation, if known

## Data Handling

This repository must not contain real company data, private user data, production endpoints, credentials, API keys, tokens, or secrets. Demo events should use safe example data only.

## Production Use

Before connecting SathiSOC Lite to real telemetry, add authentication, authorization, audit logging, input validation, rate limiting, secure storage, and a clear data retention policy.
